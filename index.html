About F# and fsharpConf

F# is an open-source, cross-platform, functional first programming language. F# makes it easy to solve complex problems with simple code. It integrates smoothly with existing .NET code bases. And... it makes coding fun again!

F# has an active open-source community building on a wide range of tools for web programming, data analytics, testing, cloud computing and much more. At fsharpConf, we want to bring some of the community together in a single-day online event!

The fsharpConf conference is organized by the F# community. The event is the third edition, following a successful fsharpConf in 2018 and 2016. It was originally inspired by dotNetConf and we are excited to be continuing this great tradition of virtual events!

Probabilistic Programming: The What, Why and How

f(x) = y, but y = 5. What’s the probability that x =1? Probabilistic programming is the idea of describing probabilistic models as programs, to then automatically infer how our beliefs about model parameters change given observed data. In recent years, probabilistic programming languages (such as Stan) have demonstrated the power of this approach by becoming the underlying tool behind numerous projects in social science, biology, genetics, astrophysics, and engineering.

But why haven’t such languages been more widely adopted yet? In this talk, Maria will talk about the nuts and bolts of probabilistic programming languages, addressing the challenges behind making these languages general-purpose, automatic, and efficient. In addition, she will discuss how ideas from programming-language research can be adopted in probabilistic systems, and how F# has helped her in the endeavour of bridging the gap between statistical modelling and probabilistic inference.

Maria Gorinova

Maria is a PhD Student at the University of Edinburgh with broad interests range from programming languages and verification, to machine learning and human-computer interaction. More specifically, she's interested in probabilistic programming languages, and in exploring ways of applying program-analysis techniques to existing PPLs in order to improve usability of the language or efficiency of inference. During her PhD, she has been working on these topics in relation to the probabilistic programming languages Stan and Edward2.

F# and Ham Radio

How I used F# and .NET Core to develop a customizable APRS messaging system, for digital radio communications, that runs on a Raspberry Pi and inexpensive radio equipment. I will focus on the functional domain modeling aspects, which include designing by types and using active patterns to parse messages. My project is called Chickadee, is open source, and you can find it on GitHub. Features include a web UI built on Saturn, a map using Azure Maps, a background service, a CLI built with Argu, a test suite built on Expecto, and a build script using FAKE.

F# Domain Modeling and Mathematical Planning

Do you or your clients ever have the question, "What would be the best course of action for us to take considering the options we have available and the restrictions we are under?" This is the question that businesses and clients are asking every day, sometimes multiple times a day. The good news is that there is an entire branch of Mathematics, Mathematical Planning, which focuses on this question and it is easier for developers to harness these tools than ever before.

Mathematical Planning is a powerful tool for decision making but has largely been relegated to use by experts. Recently it has been breaking out of its isolation due to great libraries becoming available for Python (PuLP and the Gurobi Python Library). Now the challenge is making sure your model is correctly formulated since Math cannot tell you if what you wrote makes sense.

Fortunately, F#s powerful type system makes creating a Domain Model for your problem simple and straightforward. We will show that the F# type system and Mathematical Planning are a beautiful marriage and make asking the question, "What is the best plan?" a joy.

Matthew Crews

Matthew Crews is a Machine Learning Engineer who focuses on the intersection of Mathematical Optimization and Predictive Models. He loves to teach and build tools to make it easier for developers to harness the power of Mathematical Optimization.
